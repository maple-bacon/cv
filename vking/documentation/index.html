Colors

By default, 'V-King' has a global color - yellow ( FFD600 ). Also, as you saw in the demonstration included fife color themes - Blue( 0BA4D6) , Red ( FF6347 ),Green(4ED0AA). To change the color, you must do the following steps:
1. Locate the folder css file style.css and open it with your text editor.
2.In this file, you should find a line responsible for the color theme. Look at the code below.
3.If you want to use the colors presented in this topic - In the folder "color-style" open the file and copy the code to the color you want, and replace it in the file "style.css".
4. If you want to use your color - just replace the six numbers and letters after the lattice in the file "style.css". Forgot to mention the theme color using hex format. If you want to add color format "rgba" then need to remove the lattice. If you do not know where to find the code of the color you click on this link.

Subscribe Form

In this case, you do not need to change anything

To see people who have signed up, simply open the file users-mail.txt . It is located in the folder php . There you will find all the signatories of people email

Server side script for sending emails is in "subscription.php" . Ajax submit is initialized in "scripts.js" file. 


// Subscribe   ----------------------------------------

	$('.subscriptionForm').submit(function(){
		var email = $('#subscriptionForm').val();
		$.ajax({
			url:'php/subscription.php',
			type :'POST',
			dataType:'json',
			data: {'email': email},success: function(data){
				if(data.error){
					$('#error').fadeIn()
				}
				else{
					$('#success').fadeIn();
					$("#error").hide();}
				}
			});
		return false
	});

	$('#subscriptionForm').focus(function(){
		$('#error').fadeOut();
		$('#success').fadeOut();
	});

	$('#subscriptionForm').keydown(function(){
		$('#error').fadeOut();
		$('#success').fadeOut();
	});

How to find the coordinates of Google maps ?

1. Go to Google maps
2. LEFT-click on the place that you want coordinates for.
3. Notice that there is a small circle which radiates at that place, or the nearest on-street place.
4. In the top left of the map, a small display box appears, showing the address and lat/long coordinates

You can copy / paste these as text.


if you have all the same did not happen to get the coordinates, click on this link. And by simply clicking on the map in the first top form you receive the coordinates

Seo

In this section, I will discuss the basic search engine optimization of your website .

In seo folder - you can see two files robots.txt and sitemap.xml. It first looks for files that when a search bot visits your website. Copy these files to the root of your site.

Next, open the file first robots.txt . And it set the path to the file sitemap.xml . For example: 

User-agent: *
Disallow: /css
Sitemap: http://yourdomain.com/Sitemap.xml

Next, open the file Sitemap.xml . And it prescribes all the links that exist on our website . For example: 






    
      http://yourdomain.com/about.html
      daily
    
    
      http://yourdomain.com/team.html
      daily
